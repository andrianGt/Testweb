Tentang Saya

Kenali lebih dekat pengembang di balik add-on dan patch CraftForge Studio.

Halo! Saya adalah seorang developer independen dan kreator di balik ekosistem add-on Minecraft Bedrock sejak tahun 2021.

Berawal dari hobi memodifikasi file game untuk dimainkan bersama teman, saya mulai mendalami pembuatan model 3D di Blockbench hingga pemrograman Scripting API menggunakan JavaScript.

Fokus utama saya adalah menghadirkan add-on berkualitas tinggi, stabil, optimal untuk perangkat seluler, serta bebas dari safelink iklan yang mengganggu agar nyaman diunduh oleh siapa saja.
